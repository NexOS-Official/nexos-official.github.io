### BYOD Link creation step-by-step:
* Join the Discord
* Go to https://freedns.afraid.org
* **GO TO THE REGISTRY PAGE**
* Search for your desired domain
* Once selected, make sure it is set to TYPE: A
* Under `subdomain` type anything you want
* For destination, enter the Ghost/Nexos IPv4 address ON THE DISCORD IN THE `「😊」byod-how-to` CHANNEL
* Do the Captcha and click `Save`
* Go over to the `「💖」commands` channel & run !adddomain https://THE.DOMAIN.YOU-MADE-ON-FREE-DNS. (MAKE SURE YOU ALWAYS HAVE THE `https://`)
* Wait 5 minutes-24 hours for the link to work.
* ENJOY!
